# 机器学习核心知识点

## 1. 学习范式

### 监督学习（Supervised Learning）
有标注数据，学习特征 X 到标签 Y 的映射：
- **回归**：标签连续值（房价预测）
- **分类**：标签离散值（邮件垃圾分类）

### 无监督学习（Unsupervised Learning）
无标注，发现数据内在结构：
- **聚类**：K-Means、DBSCAN、层次聚类
- **降维**：PCA、t-SNE、UMAP
- **生成模型**：VAE、GAN

### 强化学习（Reinforcement Learning）
Agent 通过与环境交互，根据奖励信号学习策略：
- **Q-Learning**、**Policy Gradient**、**Actor-Critic**
- 应用：游戏 AI（AlphaGo）、机器人控制、推荐系统探索

## 2. 经典算法

### 线性回归（Linear Regression）
模型：$\hat{y} = w^T x + b$  
损失函数：MSE（均方误差）$L = \frac{1}{n}\sum(y_i - \hat{y}_i)^2$  
求解：最小二乘法（解析解）或梯度下降

### 逻辑回归（Logistic Regression）
二分类，将线性输出通过 Sigmoid 映射到 [0,1]：  
$P(y=1|x) = \sigma(w^T x + b) = \frac{1}{1+e^{-z}}$  
损失函数：交叉熵（Cross-Entropy Loss）  
**优点**：可解释性强、训练快  
**缺点**：线性决策边界，无法处理非线性问题

### 决策树（Decision Tree）
递归地用最优特征划分数据：
- **划分准则**：信息增益（ID3）、基尼系数（CART）、信息增益率（C4.5）
- **基尼系数**：$Gini = 1 - \sum_k p_k^2$
- **剪枝**：预剪枝（提前停止）、后剪枝（剪去不重要的子树减少过拟合）

### 随机森林（Random Forest）
Bagging + 随机特征选择的决策树集成：
- 每棵树用有放回采样（Bootstrap）的子集训练
- 训练时每次分裂只考虑随机 $\sqrt{m}$ 个特征
- 预测：分类取多数投票，回归取均值
- **优点**：鲁棒，不易过拟合，重要性评估

### 支持向量机（SVM）
寻找最大间隔超平面，间隔 $= \frac{2}{\|w\|}$：
- **硬间隔**：线性可分，最大化间隔（无分类错误）
- **软间隔**：允许少量误分类（C 参数控制）
- **核函数**：RBF 核、多项式核处理非线性（特征空间映射）
- **支持向量**：距离决策边界最近的样本点

## 3. 神经网络基础

### 反向传播（Backpropagation）
链式法则求梯度，从输出层向输入层逐层传播误差，更新参数。

### 激活函数
| 函数 | 公式 | 特点 |
|---|---|---|
| Sigmoid | $1/(1+e^{-x})$ | 输出 [0,1]，梯度消失 |
| Tanh | $(e^x-e^{-x})/(e^x+e^{-x})$ | 输出 [-1,1]，零中心 |
| ReLU | $\max(0, x)$ | 计算快，负数梯度为 0（神经元死亡） |
| LeakyReLU | $\max(0.01x, x)$ | 解决神经元死亡 |
| GELU | 高斯误差线性单元 | Transformer 常用 |

### 优化器
- **SGD**：随机梯度下降，可能震荡。
- **Momentum**：累积历史梯度方向，加速收敛。
- **Adam**：自适应学习率，一阶矩（动量）+ 二阶矩（梯度平方均值）。
- **AdamW**：Adam + 权重衰减分离，Transformer 标配。

## 4. CNN / RNN / Transformer

### CNN（卷积神经网络）
- **卷积层**：局部感受野 + 权重共享，提取局部特征（平移不变性）。
- **池化层**：Max Pool 降维，减少参数。
- **应用**：图像分类（ResNet）、目标检测（YOLO）、NLP（TextCNN）。

### RNN / LSTM
- **RNN**：序列数据，有隐藏状态 $h_t = f(h_{t-1}, x_t)$，存在梯度消失/爆炸。
- **LSTM**：引入遗忘门、输入门、输出门，解决长期依赖问题。
- **GRU**：LSTM 简化版，合并遗忘门和输入门，效果相近。

### Transformer
核心：**自注意力机制（Self-Attention）**  
$$\text{Attention}(Q,K,V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

- **多头注意力**：多组 Q/K/V 并行，捕获不同子空间语义。
- **位置编码**：正弦/余弦编码注入序列位置信息。
- **优点**：并行训练（vs RNN 串行），长距离依赖捕获能力强。
- **代表模型**：BERT（双向，预训练理解）、GPT（单向，生成）。

## 5. 过拟合与正则化

### 识别过拟合
- 训练集损失低，验证集损失高（泛化差距大）。
- 学习曲线：训练曲线下降，验证曲线反弹。

### 正则化方法
| 方法 | 原理 | 适用 |
|---|---|---|
| L1 正则（Lasso） | 加 $\lambda\|w\|_1$，稀疏解（特征选择） | 高维稀疏特征 |
| L2 正则（Ridge） | 加 $\frac{\lambda}{2}\|w\|_2^2$，权重趋小 | 通用 |
| Dropout | 训练时随机丢弃神经元 | 深度网络 |
| Batch Normalization | 标准化层输入，稳定训练 | 深度网络 |
| 数据增强 | 扩充训练数据（翻转/裁剪） | 图像、NLP |
| Early Stopping | 验证损失不再下降时停止训练 | 通用 |

## 6. 评价指标

### 分类任务
- **混淆矩阵**：TP / TN / FP / FN
- **Accuracy**：$\frac{TP+TN}{总数}$（类别不平衡时无意义）
- **Precision（精确率）**：$\frac{TP}{TP+FP}$，预测为正中有多少真正是正
- **Recall（召回率）**：$\frac{TP}{TP+FN}$，真正的正中有多少被找出来
- **F1**：$\frac{2 \cdot P \cdot R}{P+R}$，精确率和召回率的调和均值
- **AUC-ROC**：不同阈值下的 TPR vs FPR 曲线面积，类别不平衡下优于 Accuracy

### 回归任务
- **MSE**：均方误差，对大误差敏感
- **MAE**：绝对误差均值，鲁棒
- **RMSE / R²**：RMSE 同单位可解释，R² 衡量拟合优度

### 排序/推荐任务
- **NDCG@K**（归一化折损累计增益）：考虑排名位置的质量
- **MAP**（平均精确率均值）：多次查询下精确率均值
- **MRR**（均值倒数排名）：第一个相关结果的排名倒数均值
